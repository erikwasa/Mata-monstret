# Mata-monstret

Mata monstret det han vill ha.

## Röstfiler och TTS

Röstfraserna finns samlade i `src/voice-lines.js` och arbetsflödet finns beskrivet i `docs/tts-workflow.md`.

Vanliga kommandon:

```bash
npm run voice:list
npm run voice:missing
npm run voice:tts-input > audio/voice-lines-missing.json
```
